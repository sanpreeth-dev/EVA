# package.json

This file defines the project's metadata, available scripts, and all npm dependencies required for the frontend.

## Scripts
- `npm run dev`: Starts the Vite development server.
- `npm run build`: Compiles the project into optimized production-ready files.
- `npm run preview`: Previews the locally built production files.

## Key Dependencies
- **React & ReactDOM**: The core library for the user interface.
- **Three.js & React Three Fiber/Drei**: Powers the 3D avatar and the underlying WebGL rendering engine.
- **GSAP & Framer Motion**: Used for smooth animations and transitions across the UI.
- **Socket.IO Client**: Facilitates real-time communication with the backend.
- **Tailwind CSS**: The utility-first CSS framework used for rapid styling.
- **Lucide React**: Provides a set of consistent icons for the interface.
