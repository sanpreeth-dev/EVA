# index.css

The global stylesheet for the application.

## Contents
- **Fonts**: Imports the "Inter" font family from Google Fonts.
- **Tailwind Directives**: Injects Tailwind's base, components, and utilities layers.
- **Global Styles**: 
    - Sets the base background to a signature purple gradient.
    - Defines a `greenScreen` class for the body, which is used when the user toggles the Green Screen mode in the 3D scene.
    - Forces the root container to occupy the full viewport (`100vw`, `100vh`).
