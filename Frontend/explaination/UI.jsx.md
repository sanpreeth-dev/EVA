# UI.jsx

The 2D interface layer that floats on top of the 3D canvas.

## Key Features
- **Karaoke Lip-Sync**: A sophisticated sub-component that highlights text word-by-word as the avatar speaks, calculated using real-time audio progress.
- **Intro Logic**: Manages the first-time greeting behavior, ensuring the avatar waves and speaks automatically once per session.
- **Camera Toggle**: Controls the zoom state of the 3D camera.
- **Image Analysis**: Includes a button to open the camera modal, allowing users to share their current mood/environment with EVA.
- **Chat Interface**: A glassmorphic input bar for text-based conversations.
