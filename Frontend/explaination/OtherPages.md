# OtherPages.md

This file summarizes the secondary pages in the `src/pages` directory.

- **Login.jsx / Signup.jsx**: Professional auth forms featuring glassmorphism, input validation, and password visibility toggles. They interact with the Backend API to manage user sessions.
- **UserGuide.jsx**: An interactive walkthrough explaining how to use EVA, featuring step-by-step cards and clear CTA buttons.
- **About.jsx**: Provides background information on the project and the mission of creating an "AI that cares."
- **ErrorPage.jsx**: A friendly 404/Error handler that matches the application's aesthetic, allowing users to safely navigate back home if something goes wrong.
