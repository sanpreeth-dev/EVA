# Welcome.jsx

The high-impact landing page of the application.

## Key Features
- **Hero Section**: Features dynamic text with a "gradient move" animation and a hover-revealing acronym explanation for "EVA" (Emotional Virtual Assistant).
- **GSAP Animations**: Uses `useGSAP` for punchy entrance animations when the page first loads.
- **Framer Motion**: Adds subtle floating effects to background circles and interactive scales to buttons.
- **Navigation**: Provides direct links to the `/avatar` (AI interaction) and `/userGuide` pages.
- **Social Proof**: Includes stylized feature cards and testimonials to build user trust.
