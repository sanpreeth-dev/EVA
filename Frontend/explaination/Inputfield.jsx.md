# Inputfield.jsx

A reusable, styled input component used throughout the Auth pages (Login/Signup).

## Features
- **Password Visibility**: Includes a built-in toggle to show/hide password text using Lucide icons.
- **Dynamic Icons**: Accepts an `Icon` component as a prop to display specialized branding for each field (e.g., Mail, Lock, User).
- **Glassmorphic Styling**: Consistent with the project's aesthetic, featuring translucent backgrounds and subtle borders.
