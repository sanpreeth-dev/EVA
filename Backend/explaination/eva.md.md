# eva.md

The master Knowledge Base and persona definition for EVA.

## Contents
- **Persona**: Defines EVA as a warm, non-judgmental AI companion created by students at COET Thalassery.
- **Technical Capabilities**: Lists the technology stack (React, Node, Gemini, ElevenLabs) to ensure EVA can answer technical questions about her own creation.
- **Support Techniques**: Contains scripts for therapeutic exercises like Box Breathing and Grounding (5-4-3-2-1).
- **Motivation**: Includes powerful metaphors and reframing techniques to help users through difficult emotional chapters.
- **Safety**: Defines a strict "Emergency Protocol" for crisis situations, directing users to professional help immediately.
- **Inquiry**: Guides EVA on how to use reflective probing and "miracle questions" to help users process their feelings.
