# EVA

## Emotional Virtual Assistant

**EVA** is a web-based emotional support platform featuring a 3D virtual  avatar that interacts with users through realistic voice, expressions, and emotionally aware responses. Designed to provide supportive conversations for stress management and emotional wellness, EVA bridges the gap for users who lack immediate access to human mental health support.

* **Real-time Emotion Detection:** Integrates facial emotion recognition via webcam and machine learning to adapt the avatar's behavior to the user's mood.
* **3D Interactive Avatar:** Utilizes **Ready Player Me** and **Three.js** to create a lifelike, human-centric interface.
* **Empathetic Conversation:** Powered by **Google Gemini**, the assistant handles dialogue with high emotional intelligence and empathy.
* **Advanced Lip-Sync & Voice:** Features **ElevenLabs** for high-quality voice synthesis and **Rhubarb Lip Sync** for natural avatar mouth movements.

### **Frontend**
* **React.js:** Core framework for building a dynamic, responsive user interface.
* **Three.js:** Used for rendering and animating the 3D virtual  avatar.
* **Tailwind CSS:** For sleek, modern, and mobile-friendly styling.
* **Vite:** High-performance build tool for a fast development experience.

### **Backend**
* **Node.js & Express:** Scalable server-side architecture to manage API requests and avatar logic.
* **MongoDB:** NoSQL database utilized for storing user history and as a vector store for RAG (Retrieval-Augmented Generation) implementation.
* **OpenCV:** Integrated for processing webcam feeds to perform facial emotion detection.

### **AI & ML**
* **TensorFlow/Keras:** Custom-trained CNN models for high-accuracy emotion classification.
* **Gemini API:** State-of-the-art LLM processing for natural language understanding with a focus on emotional awareness.
